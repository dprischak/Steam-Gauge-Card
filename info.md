# Steam Gauge Card

A custom Lovelace card for Home Assistant that displays sensor values as a vintage steam engine gauge with an aged, industrial aesthetic.

## Features

- Vintage steam engine aesthetic with aged brass and worn appearance
- Animated flip-style digital display (odometer effect)
- Configurable color segments and ranges
- Customizable min/max values and angle ranges
- Smooth needle animations
- Multi-line title support
- Visual configuration editor included

Perfect for displaying temperature, humidity, power consumption, pressure, or any numeric sensor data with a retro industrial style.
